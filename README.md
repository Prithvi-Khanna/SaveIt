Currently this app runs perfectly on iOS

It runs with a json server as a backend
Base Url ==> 'http://localhost:3001'

Steps to run the App
1. Clone the App
2. npm install
3. cd ios && pod install

run the json server with commands in separate terminal --> 
4.  npm install -g json-server
5. json-server --watch db.json --port 3001     

6. npm run ios
